High-Level Architecture
Browser ← HTTP → Next.js Server (port 3000)
├── Frontend: Pages Router / _app.tsx
│ ├── Layout (MUI Container)
│ │ ├── AppHeader (AppBar + icon + title)
│ │ ├── NotesSidebar (Drawer — note list)
│ │ └── MainArea
│ │ └── TabBar + TabPanel (TipTap editor)
│ └── Context: NoteContext, TabContext
└── API Routes: /api/notes/*
└── MongoDB Driver → MongoDB (port 27017)
