MUI Component Mapping

Section 2/3 — Every UI element mapped to MUI

MUI Components Used
ElementMUI Component
Header barAppBar + Toolbar + Typography
Notes iconIcon (Notes icon from MUI icons)
SidebarDrawer (permanent variant)
Note list itemsList + ListItem + ListItemButton
TabsTabs + Tab (scrollable)
New Note buttonFab or Button (with + icon)
Delete noteIconButton with DeleteIcon
Tab closeIconButton with CloseIcon (inside Tab label)
Rich text toolbarToggleButtonGroup (Bold/Italic/Underline)
Font selectorSelect dropdown
Confirm dialogsDialog + DialogTitle + DialogActions

MUI Theme

Default MUI theme with a custom primary color (blue/grey). We'll use the built-in createTheme in _app.tsx or a ThemeProvider wrapper. Light mode only for now — dark mode can be added later.
